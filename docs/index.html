Документация TGBOT SOLAR API

Документация для плагина Telegram API для Solar2D, созданного @xqss_DEVELOPER. Поддерживает ботов (Bot API) и юзерботов (MTProto API).

Установка

Скопируйте telegram_api.lua в корень проекта.

Для ботов: получите токен от @BotFather.

Для юзерботов: получите api_id и api_hash на my.telegram.org.

local telegram = require("telegram_api")

-- бот
telegram.initBot({
    token = "YOUR_BOT_TOKEN",
    botName = "@YourBotName"
})

-- юзербот
telegram.initUserbot({
    apiId = 12345,
    apiHash = "YOUR_API_HASH",
    phoneNumber = "+1234567890"
})

Константы

Константы заменяются данными из последнего события.

  • $first_name$: Имя пользователя.
  • $last caras="mb-2 mt-2">2. Всплывающее уведомление:

    telegram.answerCallbackQuery(callbackQueryId, "действие подтверждено, $username$!", true, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("уведомление отправлено!")
    end
end)

    3. Подтверждение без текста:

    telegram.answerCallbackQuery(callbackQueryId, nil, false, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("запрос подтвержден!")
    end
end)

    4. Обработка в polling:

    local function eventHandler(eventType, data)
    if eventType == "callback_query" then
        if data.data == "btn1" then
            telegram.answerCallbackQuery(data.callbackQueryId, "кнопка 1, $first_name$!", true)
        else
            telegram.answerCallbackQuery(data.callbackQueryId, "другое, $username$", false)
        end
    end
end

-- запускаем polling
telegram.startBotPolling(eventHandler)

API для Юзерботов

API для юзерботов использует MTProto API, позволяя выполнять действия от имени пользователя. Внимание: Юзерботы неофициальны, использование VOIP-номеров может привести к бану.

Инициализация и Авторизация

Инициализируйте юзербота и авторизуйтесь.

telegram.initUserbot({
    apiId = 12345,
    apiHash = "YOUR_API_HASH",
    phoneNumber = "+1234567890"
})

-- запрос кода
telegram.requestCode(function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("код отправлен!")
    end
end)

-- вход
telegram.signIn({
    code = "12345",
    password = "your_2fa_password"
}, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("авторизация успешна!")
    end
end)

Параметры:

  • apiId, apiHash, phoneNumber: Обязательные.
  • code, password: Для signIn.

Методы Юзербота

1. Отправка сообщения:

telegram.userbotSendMessage(nil, "привет, $first_name$!", nil, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("сообщение отправлено!")
    end
end)

2. Отправка медиа:

telegram.userbotSendMedia(nil, "photo", "https://example.com/image.jpg", {
    caption = "фото для $first_name$!"
}, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("медиа отправлено!")
    end
end)

3. Получение участников чата:

telegram.userbotGetChatMembers(123456789, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        for _, member in ipairs(result.members) do
            print("участник: " .. member.first_name)
        end
    end
end)

4. Пересылка сообщения:

telegram.userbotForwardMessage(123, 456, nil, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("сообщение переслано!")
    end
end)

5. Редактирование сообщения:

telegram.userbotEditMessage(nil, 123, "новый текст", function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("сообщение отредактировано!")
    end
end)

6. Удаление сообщения:

telegram.userbotDeleteMessage(nil, 123, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("сообщение удалено!")
    end
end)

7. Создание чата:

telegram.userbotCreateChat("новый чат", {123, 456}, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("чат создан!")
    end
end)

8. Присоединение к каналу:

telegram.userbotJoinChannel(123456789, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("присоединился к каналу!")
    end
end)

9. Выход из канала:

telegram.userbotLeaveChannel(123456789, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("покинул канал!")
    end
end)

10. Приглашение в чат:

telegram.userbotInviteToChat(nil, {123, 456}, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("пользователи приглашены!")
    end
end)

11. Кик из чата:

telegram.userbotKickChatMember(nil, 123, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("пользователь исключен!")

end)

12. Получение истории чата:

    if error then
        print("ошибка: " .. error)
    else
        print("получено сообщений: " .. #result.messages)
    end
end)

13. Поиск сообщений:

    if error then
        print("ошибка: " .. error)
    else
        print("найдено сообщений: " .. #result.messages)

end)

14. Установка статуса ввода:

telegram.userbotSetTyping(nil, function(result, error)
    if error then
        print("ошибка: " .. error)
    else
        print("статус ввода установлен!")

15. Получение информации о пользователе:
