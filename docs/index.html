Документация TGBOT SOLAR API

Документация для плагина Telegram API для Solar2D, созданного @xqss_DEVELOPER. Поддерживает ботов (Bot API) и юзерботов (MTProto API).

Установка

Скопируйте telegram_api.lua в корень проекта.

Для ботов: получите токен от @BotFather.

Для юзерботов: получите api_id и api_hash на my.telegram.org.

local telegram = require("telegram_api")

-- Инициализация бота
telegram.initBot({
    token = "YOUR_BOT_TOKEN",
    botName = "@YourBotName"
})

-- Инициализация юзербота
telegram.initUserbot({
    apiId = 12345,
    apiHash = "YOUR_API_HASH",
    phoneNumber = "+1234567890"
})

Константы

Константы заменяются данными из последнего события.

  • $first_name$: Имя пользователя.
  • $last_name$: Фамилия пользователя.
  • $age$: Возраст пользователя (если указан).
  • $last_message_text$: Текст последнего сообщения.
  • $username$: Имя пользователя (@username).
  • $chat_id$: ID чата.

API для Ботов

API для ботов использует telegram_api.lua и Telegram Bot API.

Инициализация

    botName = "@YourBotName"
})

        print("Сообщение отправлено!")

API для Юзерботов

API для юзерботов использует MTProto через заглушку lua-telegram-mtproto.

Инициализация и авторизация

telegram.initUserbot({
    apiId = 12345,
    apiHash = "YOUR_API_HASH",
    phoneNumber = "+1234567890"
})

telegram.requestCode(function(result, error)
    if error then
        print("Ошибка: " .. error)
    else
        print("Код отправлен!")
    end
end)

telegram.signIn({
    code = "12345",
    cloudPassword = "YOUR_CLOUD_PASSWORD" -- Облачный пароль для двухфакторной аутентификации(если есть)
}, function(result, error)
    if error then
        print("Ошибка: " .. error)
    else
        print("Авторизация успешна!")
    end
end)
